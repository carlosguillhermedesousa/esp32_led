📌 Descrição do Projeto

Este projeto tem como objetivo demonstrar a integração entre um microcontrolador ESP32 e a plataforma IoT Blynk, permitindo o controle remoto de um LED através de um aplicativo mobile.

A aplicação foi desenvolvida utilizando a linguagem C++ no ambiente Arduino, explorando conceitos de Internet das Coisas (IoT), comunicação em rede e automação básica.

🚀 Funcionalidades
📲 Controle remoto de LED via aplicativo Blynk
🔄 Comunicação em tempo real entre o ESP32 e o app
⏱️ Acionamento temporizado (LED ligado por 500ms)
📊 Atualização de status no aplicativo (feedback visual)
🌐 Conexão via rede Wi-Fi
🛠️ Tecnologias Utilizadas
ESP32
Plataforma Blynk
Arduino IDE
Protocolo Wi-Fi
📚 Objetivo Educacional

Este projeto foi desenvolvido como uma atividade prática aplicada em ambiente educacional, com foco no aprendizado de conceitos fundamentais de IoT, programação embarcada e integração entre hardware e software.

Indicado para alunos iniciantes em desenvolvimento de sistemas embarcados e automação, proporcionando uma experiência prática com controle remoto de dispositivos físicos.

⚙️ Funcionamento

O usuário aciona um botão no aplicativo Blynk (V0), que envia um comando para o ESP32. Ao receber esse comando:

O LED onboard é ligado
O status é atualizado no aplicativo (V1)
Após 500 milissegundos, o LED é automaticamente desligado
O status é novamente atualizado no app
